1

光源类型

四种光源各自的特点、方向性、阴影特性

⚡ 基础概念，与 M09 阴影联动

Ambient Light
环境光

无方向性，均匀来自所有方向，模拟间接光照。无位置、无方向，对场景所有点贡献相同。

无阴影 Ia = ka·Ic

Point Light
点光源

从一点向所有方向发出光线。近处光源光线发散，远处可近似为平行光。

硬阴影 各点入射角不同

Area Light
面光源

占据有限的一维或二维空间区域。可被物体部分遮挡，自然产生半影区域。

软阴影 半影 Penumbra

Spotlight
聚光灯

在特定方向上发出近似平行光束，方向性强。类似舞台聚光灯效果。

方向性强 锥形光束

考试关键区分：
· 点光源 → 硬阴影（Hard Shadow），只有本影（Umbra）
· 面光源 → 软阴影（Soft Shadow），有本影 + 半影（Penumbra）
· 平行光源（Directional Light）是点光源在无穷远处的特例，对同一平面上各点入射角相同
2

Phong 光照模型（三分量总览）

I = Ia + Id + Is — 光照 = 环境 + 漫反射 + 镜面反射

⚡ 必考：完整公式 + 各分量含义
📌 Phong 光照模型将光照分解为三个分量之和，模拟物体在光照下的视觉外观。这是计算机图形学中最经典的光照模型。
Phong 光照模型 — 单光源完整公式：
I = ka · Ic + kd · IL · max(0, N·L) + ks · IL · max(0, V·R)n
    ↑ 环境光 Ambient      ↑ 漫反射 Diffuse              ↑ 镜面反射 Specular
多光源扩展公式：
I = ka Ic + Σi=1N [ kd IL,i max(0, N·Li) + ks IL,i max(0, V·Ri)n ]

环境光 Ambient

与表面方向无关，与视点无关。常数背景光，保证物体不完全黑暗。

漫反射 Diffuse

与视点无关。光向所有方向均匀散射，亮度只取决于入射角 N·L。

镜面反射 Specular

与视点有关。高光随视点移动而移动，n 控制高光大小。

⚠️ 终极考试区分点："漫反射与视点无关（不管从哪个方向看都一样）"、"镜面反射与视点有关（高光随视点移动）"——这两句话几乎每道 Phong 相关题都会涉及，必须写到卷子上。
3

环境光（Ambient Reflection）

最简单的分量——常数背景光，保证场景不完全黑暗

⚡ 公式最简单，必须拿分
环境光公式：
Ia = ka · Ic

ka：表面环境反射系数（0 < ka < 1）
Ic：全局环境光强度（常数，来自所有方向）
特点：与表面方向无关 · 与视点无关 · 无方向性
📌 环境光是一个粗糙但必要的近似。在真实世界中，光在场景中多次弹射产生间接光照。Phong 模型用常数环境光项模拟这种效果，避免物体背面完全黑暗。
系数 ka含义视觉
ka ≈ 0完全不反射环境光阴影区纯黑
ka = 0.2微弱反射环境光阴影区微亮
ka = kd环境反射 = 漫反射系数（常见简化）柔和过渡
4

漫反射（Diffuse Reflection · Lambert's Law）

与视点无关！亮度只取决于表面法向量与光线方向的夹角

⚡ 必考：平行光 vs 点光源区分
漫反射公式（Lambert's Law）：
Id = kd · IL · cos θ = kd · IL · max(0, N · L)

kd：表面漫反射系数
IL：光源强度
θ：入射角（表面法向量 N 与光线方向 L 的夹角）
N, L：归一化向量（N·L = cos θ）

注意：N·L 可能为负（光从背面照射），需 max(0, N·L) 截断
特点：与视点方向无关（向任意方向均匀散射）
入射角 θ = 0° N·L = 1，光线垂直照射表面 → 最亮（Id = kd·IL）
入射角 θ = 60° N·L = 0.5，光线斜射表面 → 亮度减半
入射角 θ ≥ 90° N·L ≤ 0，光线从背面照射 → Id = 0（max(0, neg) = 0）
平行光源 vs 点光源 —— 考试高频区分！

平行光源（Directional Light）：
· 光线方向完全相同（如太阳光）
· 对同一平面上所有点，入射角 θ 相同 → IdA = IdB

点光源（Point Light）：
· 光线方向发散（如台灯）
· 对同一平面上不同位置的点，入射角 θ 不同 → IdA ≠ IdB
· 更靠近光源直射点的面更亮
⚠️ Q9a / Q9b 考试原题逻辑：同一平面上 A、B 两点——平行光源下 IdA = IdB（入射角相同）；点光源下 IdA 与 IdB 的大小取决于哪点更靠近光源正下方（入射角更小）。
5

镜面反射（Specular Reflection）

与视点有关！高光随视点移动而移动，n 控制高光大小

⚡ 必考：反射向量 + n 的作用
镜面反射公式：
Is = ks · IL · cosn φ = ks · IL · max(0, V · R)n

ks：表面镜面反射系数
n：镜面反射指数（Specular Exponent），控制高光大小
    n 越大 → 高光越小越尖锐（趋向完美镜面）
    n = ∞ → 完美镜面（perfect mirror）
φ：反射光方向 R 与观察方向 V 的夹角
R, V：归一化向量

特点：与视点方向有关（高光随视点移动而移动）
反射向量计算（必须背下来！）：
R = 2(N·L)N − L

推导：入射光 L 在 N 方向的分量为 (N·L)N，反射光 = 入射光 − 2×法向分量
验证：|R| = 1（反射向量仍为单位向量）

n 小（如 n=1）

高光范围大，边缘模糊。适合粗糙表面（如粉笔、未抛光的木材）。

n 大（如 n=100）

高光范围小，集中尖锐。适合光滑表面（如抛光金属、玻璃、塑料）。

📌 cosn φ 的几何直觉：当 φ 很小（观察方向接近反射方向），cos φ ≈ 1 → cosn φ ≈ 1，看到强高光。当 φ 略大（偏离反射方向），cos φ < 1 → cosn φ 快速衰减（n 越大衰减越快），高光消失。这就是为什么高光只在很小的角度范围内可见。
⚠️ Q9c 考试原题逻辑：同一 B 点，Viewer 1 和 Viewer 2 从不同方向观察。比较 Is1 和 Is2 的大小 → 比较 φ1（V1 与 R 的夹角）和 φ2（V2 与 R 的夹角）。夹角越小（越靠近反射方向），高光越强。
6

Flat Shading（平面着色）

最简单的着色方法——每个多边形一个颜色

⚡ 三种着色中最快但最粗糙
Step 1 计算多边形的面法向量（所有顶点共享一个法向量）
Step 2 使用 Phong 公式计算一个颜色值（在面中心或任意顶点处）
Step 3 将该颜色填充整个多边形所有像素
优点缺点
计算量最少，速度最快相邻多边形之间颜色不连续（intensity discontinuity）
实现简单多边形边界明显，视觉效果粗糙
适合光源/视点很远或多边形很小的情况完全无法呈现高光效果
7

Gouraud Shading（高洛德着色）

颜色插值——在顶点算光照，内部线性插值颜色

⚡ 最高频考点：为何丢失高光
Step 1 计算每个顶点的法向量：nvertex = normalize(Σ 相邻面法向量)
Step 2 在每个顶点处用 Phong 公式计算颜色值 Cv
Step 3 在多边形内部用线性插值得到像素颜色
📌 优点：消除相邻面颜色不连续，视觉平滑。比 Phong Shading 快，硬件广泛支持。适合漫反射材质（布料、哑光表面）。
🚫 核心缺陷 —— Gouraud 为何漏掉高光（Q7a 答案！）：

根本原因：颜色插值 vs 法向量插值

Gouraud 插值的是"颜色"（Color Interpolation），而高光（镜面反射）是非线性的：
Is = ks · IL · (V·R)n → 当 n 很大时，高光只在很小角度范围内出现

若高光的峰值恰好出现在多边形内部（而非顶点处）：
· 顶点处 V·R 较小 → 顶点颜色较暗
· 内部 V·R 最大 → 高光应出现在这里
· 但 Gouraud 是从"较暗的顶点颜色"线性插值 → 内部颜色也偏暗
· 高光消失！

若高光出现在顶点处：
· 顶点处计算出明亮的高光颜色
· 高光随视点移动（因为是在顶点处计算的），造成"游走"（Highlight Walking）效果
⚠️ 考试关键词：颜色线性插值 / 非线性高光 / 内部峰值无法捕捉 / 高光丢失（disappear）/ 高光游走（flickering / walking）。
8

Phong Shading（冯氏着色 · 法向量插值着色）

法向量插值——在每像素重新计算 Phong 光照，高光精确

⚡ Q7b 答案：插法向量非颜色
Step 1 计算每个顶点的法向量 nv（取相邻面法向量加权平均）
Step 2 在三角形内部用线性插值 / 重心坐标插值法向量，并归一化
Step 3 在每个像素处用插值后的法向量 npixel 和 Phong 公式重新计算颜色
📌 解决高光的关键：Phong Shading 不插值颜色，而是插值法向量（Normal Vector）。高光峰值无论落在三角形的哪个位置，该像素处的法向量都能正确指向高光方向 → Phong 公式在该像素计算出正确的 Is → 高光被精确捕捉。
✅ Q7b 标准答案框架：

① Phong Shading 不插值颜色，而是插值法向量（Normal Vector）
② 在三角形内部每个像素处，对顶点法向量做线性插值 / 重心坐标插值并归一化，得到每像素法向量
③ 对每个像素独立重新计算完整的 Phong Illumination Model
④ 无论高光峰值落在哪里（内部或顶点），都能被精确捕获，不依赖多边形密度
⑤ 代价：每像素都要执行完整 Phong 公式 → 计算开销远高于 Gouraud

三种着色方法对比表格

Flat vs Gouraud vs Phong — 插值对象、计算位置、高光、开销、适用场景

⚡ 高频完整三方对比
对比维度Flat ShadingGouraud ShadingPhong Shading
插值对象 无插值（每多边形一个颜色） 颜色（Color） 法向量（Normal Vector）
计算位置 每多边形一次 每顶点一次 每像素一次
高光（Specular）表现 ❌ 完全无法呈现 ❌ 高光易丢失或"游走"（非线性被线性化） ✅ 精确，无论高光在三角形哪个位置
相邻面颜色连续性 ❌ 相邻面颜色突变（边界明显） ✅ 平滑过渡 ✅ 平滑过渡
计算开销 最低（最快） 中等 最高（最慢，逐像素）

核心区别记忆口诀：
· Flat：一个多边形 → 一个颜色
· Gouraud：一个顶点 → 一个颜色 → 内部插值颜色
· Phong：一个顶点 → 一个法向量 → 内部插值法向量 → 每像素算颜色

Gouraud 失效条件：高光指数 n 很大（小面积高光）且高光峰值在三角形内部时，Gouraud 会完全漏掉高光。

考场答案模板

闭卷可直接套用的答题框架

⚡ 两套模板覆盖本模块全部题型
模板 A｜Gouraud Shading 为什么会漏掉高光（Q7a 答题框架）

适用题型："Why does Gouraud shading fail to render specular highlights correctly?"

答题框架（考场直接写）：

① Gouraud 的做法：仅在顶点处计算完整 Phong 光照（含 Specular 分量），然后在三角形内部对颜色（Color）进行线性插值。

② 问题根源：Specular Highlight 是非线性效果（Is = ks · IL · (V·R)n），当 n 较大时高光极度集中，峰值往往落在三角形内部而非顶点处。

③ 高光消失的逻辑：顶点处 V·R 较小 → 顶点颜色较暗 → 从暗色顶点做线性插值 → 内部颜色也暗 → 高光完全丢失。

④ 高光闪烁的逻辑：若高光偶然靠近顶点 → 随相机移动顶点处 V·R 剧烈变化 → 颜色不稳定 → 高光闪烁（flickering）。

关键词：颜色线性插值 / 非线性高光 / 内部峰值无法捕捉

模板 B｜Phong Shading 如何解决 Gouraud 的问题（Q7b 答题框架）

适用题型："How does Phong shading address the highlight problem in Gouraud shading?"

答题框架（考场直接写）：

① 核心差异：Phong Shading 不插值颜色，而是插值法向量（Normal Vector）。

② 流程：每个顶点取相邻面法向量的加权平均得顶点法向量 → 在三角形内部对顶点法向量做线性插值 / 重心坐标插值（需归一化）→ 用插值法向量在每个像素独立执行完整 Phong 光照计算。

③ 解决高光的原因：高光峰值无论落在三角形的哪个位置，该像素处的法向量都能正确指向高光方向 → Phong 公式在该像素计算出正确的 Is → 高光被精确捕捉。

④ 代价：每像素都要执行光照计算 → 计算开销远高于 Gouraud。

一句话对比：Gouraud 插颜色；Phong 插法向量，逐像素算颜色。

模板 C｜Phong 光照模型数值计算框架（Q9 答题框架）

适用题型："Calculate the diffuse/specular intensity given angles" / "比较两点光照强度大小"

三分量公式（必须熟记）：

① 环境光：Ia = ka · Ic（与表面方向无关，与视点无关）

② 漫反射：Id = kd · IL · max(0, N·L)（与视点无关）

③ 镜面反射：Is = ks · IL · max(0, V·R)n（与视点有关）

反射向量：R = 2(N·L)N − L

比较大小的快速逻辑：

· 比较漫反射：看 N·L（入射角越小越亮）

· 平行光源 → 同一平面各点 Id 相同；点光源 → 各点 Id 不同

· 比较镜面反射：看 V·R（观察方向越靠近反射方向越亮）

数值计算步骤：

Step 1: 算 Ia = ka × Ic

Step 2: 算 N·L → Id = kd × IL × N·L

Step 3: 算 R = 2(N·L)N − L（验证 |R| = 1）

Step 4: 算 V·R → Is = ks × IL × (V·R)n

Step 5: I = Ia + Id + Is

⚡ 含数值计算 + 概念对比
P7-1. Phong 光照模型三分量数值计算

已知场景参数：

· 环境光强度 Ic = 0.3，环境反射系数 ka = 0.2

· 光源强度 IL = 1.0，漫反射系数 kd = 0.6，镜面反射系数 ks = 0.4，高光指数 n = 16

· 表面法向量 N = (0, 1, 0)（朝上）

· 光源方向 L = (0, cos 30°, sin 30°) = (0, √3/2, 0.5)（已归一化）

· 观察方向 V = (0, 1, 0)（正上方，已归一化）

(a) 环境反射 Ia：

Ia = ka · Ic = 0.2 × 0.3 = 0.06

(b) 漫反射 Id：

N · L = (0)(0) + (1)(√3/2) + (0)(0.5) = √3/2 ≈ 0.866

Id = kd · IL · (N·L) = 0.6 × 1.0 × 0.866 ≈ 0.520

(c) 反射向量 R：

R = 2(N·L)N − L = 2·(√3/2)·(0,1,0) − (0, √3/2, 0.5)

= (0, √3, 0) − (0, √3/2, 0.5) = (0, √3/2, −0.5)

验证：|R| = √(0 + 3/4 + 1/4) = 1 ✓

(d) 镜面反射 Is：

V · R = (0)(0) + (1)(√3/2) + (0)(−0.5) = √3/2 ≈ 0.866

Is = ks · IL · (V·R)n = 0.4 × 1.0 × (0.866)16

(0.866)16 = (0.866²)⁸ = (0.75)⁸ = 0.75⁴ × 0.75⁴ ≈ 0.316 × 0.316 ≈ 0.100

Is ≈ 0.4 × 0.100 = 0.040

(e) 总光照 I：

I = Ia + Id + Is = 0.06 + 0.520 + 0.040 = 0.620

⚠️ 考试要点：反射向量公式 R = 2(N·L)N − L 必须背下来。n 较大时 (V·R)n 会迅速衰减（高光集中），计算时可用逐步平方简化。

Q7 变形. Gouraud Shading 高光问题分析

(a) 为何 Gouraud Shading 无法正确呈现高光？

Gouraud Shading 仅在顶点处计算完整 Phong 光照（含 Specular 分量），然后将顶点颜色值在三角形内部做线性插值。

由于 Specular Highlight 是非线性效果（Is = ks · IL · (V·R)n），当 n 较大时高光极度集中，高光峰值往往落在三角形内部而非顶点处。从暗色顶点做线性插值 → 内部仍暗 → 高光消失。

(b) Phong Shading 如何解决此问题？

Phong Shading 不插值颜色，而是插值法向量。在三角形内部每个像素处对顶点法向量做线性插值 / 重心坐标插值并归一化，然后对每个像素独立重新计算完整 Phong 光照。无论高光峰值落在哪里，都能被精确捕获。

代价：每像素都要执行完整 Phong 公式，计算开销远高于 Gouraud。

Q9 变形. 漫反射与镜面反射关系分析

(a) 平行光源下，同一平面上 A、B 两点 IdA 和 IdB 的关系？

平行光源光线方向完全相同，A 和 B 处的入射角 θ 完全相同 → IdA = IdB。

(b) 点光源下，IdA 和 IdB 的关系？

点光源光线发散，A 和 B 处光线方向不同，入射角 θA ≠ θB。哪点更靠近光源正下方（入射角更小）→ 哪点更亮。

(c) 点光源下 B 点朝 Viewer 1 和 Viewer 2 方向的镜面反射 Is1 和 Is2 的关系？

镜面反射 Is = ks · IL · (V·R)n，与视点有关。比较 φ1（V1 与 R 的夹角）和 φ2（V2 与 R 的夹角）：

· φ 越小（越靠近反射方向）→ cos φ 越大 → Is 越大

· 哪边更靠近理想反射方向，哪边看到更高光

附加题. 反射向量公式推导

写出反射向量 R 的完整公式并解释每一项的含义。

公式：R = 2(N·L)N − L

推导：入射光 L 在法向量 N 方向的分量为 (N·L)N。反射光 = 入射光 − 2 × 法向分量 = L − 2[(−L)·N]N = 2(N·L)N − L。

验证：反射向量仍为单位向量（|R| = 1），因为反射不改变长度。

向量方向约定：L 指向光源（从表面点出发），R 指向反射方向（从表面点出发），N 为表面外法向量。

⚡ Phong 公式速查卡

必背公式（闭卷必须能默写）

  • Phong 总公式：I = kaIc + kdILmax(0,N·L) + ksILmax(0,V·R)n
  • 环境光：Ia = ka · Ic
  • 漫反射：Id = kd · IL · max(0, N·L)
  • 镜面反射：Is = ks · IL · max(0, V·R)n
  • 反射向量：R = 2(N·L)N − L
  • 多光源扩展：环境光只加一次，漫反射+镜面按光源累加

必背区分点（几乎每道题都涉及）

  • 漫反射与视点无关（不管从哪个方向看都一样）
  • 镜面反射与视点有关（高光随视点移动）
  • 平行光源：同一平面各点入射角相同 → IdA = IdB
  • 点光源：同一平面各点入射角不同 → IdA ≠ IdB
  • n 的作用：n 越大 → 高光越小越尖锐
  • n = ∞：完美镜面

三种着色方法速记

  • Flat：每多边形一次，颜色不连续，最快
  • Gouraud：顶点算颜色 → 内部插颜色 → 高光丢
  • Phong：顶点算法向量 → 内部插法向量 → 逐像素算 → 高光正确
  • Gouraud 失效条件：n 大 + 高光在三角形内部

考场容易遗漏的细节

  • N·L 必须用 max(0, N·L) 截断（背面光照为 0）
  • R = 2(N·L)N − L，注意是减去 L 不是加
  • 插值法向量后必须 normalize
  • Phong Shading 在 片段着色器（Fragment Shader） 执行
  • Gouraud Shading 在 顶点着色器（Vertex Shader） 执行
  • 面光源 → 软阴影（有半影）；点光源 → 硬阴影（只有本影）
